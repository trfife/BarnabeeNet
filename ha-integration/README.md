# Home Assistant Integration

This directory contains the BarnabeeNet custom component for Home Assistant.

## Structure

```
custom_components/
└── barnabeenet/
    ├── __init__.py
    ├── manifest.json
    ├── config_flow.py
    ├── const.py
    └── services.yaml
```

## Installation

The component is automatically deployed to Home Assistant via the infrastructure configuration.
